## Simple Player Movement-
## Особенности-
-Простой и рабчоий 3D-контроллер для игрока
-Подходит начинающим разработчикам
-Работает через Raycast
-Рабочий на новой системе Input
## Как настроить-
- Скачайте скрипт и закинте его к себе в Uniti
- поставте любой блок добавьте в New Component часть Charaster Controller
- После этого перенсите код на игрока
## Примечания-
- Неважно как называть блок (игрока)
- Добавьте под игроком Пол (Plane Или Cube)
